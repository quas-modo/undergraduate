# Lecture09 - 指针

use CONST whenever possible 只要不改变成员变量的值，就一定要加上 const

为什么 c 和 * q 中的值不一样？

因为 compiler 把 c 直接替换成 128。但是 q 中存储的确实变成了 111。

指针自己是常量，不可以改变

不懂

(x+n-1) / n * n

class —— 访问控制默认 private

剥离出来，方便增改
